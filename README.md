# 🌌 VoidWeaver

**Физически точный 3D-симулятор галактик в браузере.** Гравитационное линзирование чёрной дыры, кеплеровская механика орбит, флоккулентные газовые рукава, голосовое управление — всё в одном самостоятельном HTML-файле, без сборки и зависимостей.

## О проекте

VoidWeaver создавался в рамках развиваемой автором **Теории Фрактального Резонанса (ТФР)** — как инструмент визуализации гипотетического дополнительного эффекта гравитации, обусловленного крупномасштабной структурой вакуума («вакуумной паутиной»). Симулятор задумывался как способ наглядно представить, как такая структура могла бы проявляться в динамике галактик: распределении тёмной материи, форме нитей и филаментов, поведении орбит на разных масштабах.

При этом проект полностью самодостаточен и полезен независимо от теоретического контекста — как:

- 🎓 **образовательная визуализация** базовой астрофизики галактик (спиральные рукава, аккреционные диски, кеплеровы орбиты, гравитационное линзирование);
- 🖼️ **источник скриншотов** и иллюстраций для презентаций, статей, обложек;
- 🖥️ **музыкальная экранная заставка** — с встроенным аудиоплеером и кинематографическими режимами камеры.

> Теория Фрактального Резонанса — авторская теоретическая концепция, не являющаяся частью общепринятой физики. Сам же симулятор использует строгие физические модели там, где это возможно (метрика Шварцшильда, уравнение Кеплера, доплеровское смещение), и не претендует на экспериментальное подтверждение ТФР — это инструмент визуализации идеи, а не научное доказательство.

## ✨ Возможности

### Галактика
- Два типа: **спиральная** и **эллиптическая**, с независимой настройкой каждой
- Спиральные рукава: grand design / пересечённая / **флоккулентная** структура (фрактальный шум + смещение цветового баланса к пылевым сгусткам)
- Регулировка толщины диска — от плоского профиля до полного сфероида, для обоих типов галактик
- Нити тёмной материи: геометрия Фибоначчи / мембрана, натяжение, количество
- Газовые облака: плотность, неоднородность, флоккулентность, направление вращения

### Чёрная дыра и центральная область
- Аналитическая модель гравитационного линзирования с маской горизонта событий
- Аккреционный диск с доплеровским усилением яркости и температурным градиентом
- Фотонное кольцо, диффузное свечение, гравитационные волны от внешнего края диска
- Газо-пылевое облако из вихревых нитей с турбулентными трассерами на границе
- Случайные пролёты астероидов и комет с диффузными «тающими» хвостами

### Звёзды и орбиты
- Кеплеровская механика орбит звёзд гало: высокий эксцентриситет, сферическое распределение наклонений, ~30% ретроградных орбит — без общего вращения как у системы
- То же — для звёзд эллиптической галактики (хаотичное «роевое» движение)
- Опциональная трассировка орбитальных следов в режиме ручной перемотки времени

### Камера и кинематография
- Режимы автооблёта: Рукава / Ядро / Гало / Эпик / Туманность / **Вертиго** (Dolly Zoom)
- Slow Motion со плавным speed ramping в видеорежиме
- Lens Flare с физически обоснованными триггерами (выравнивание камеры с диском, вспышка джета)

### Управление
- Голосовое управление (локальные команды + сложные запросы через Gemini API)
- Полная русская/английская локализация интерфейса
- Адаптация под сенсорные экраны (смартфоны, планшеты)
- Скриншоты (одиночные и серийные)

### Аудио
- Встроенный плеер: файлы, папка, прямые URL или встроенный плейлист
- Озвучивание событий и атмосферное музыкальное сопровождение

Проект — один HTML-файл. Никакой установки или сборки не требуется.

## 🎛️ Управление

| Действие | Клавиша / жест |
|---|---|
| Вращение камеры | ЛКМ + перетаскивание |
| Панорама | ПКМ + перетаскивание |
| Зум | Колесо мыши |
| Перемотка времени | Alt + колесо |
| Пауза / пуск | Ctrl |
| Видеорежим (AUTO) | Enter |
| Полный экран | F |
| Голосовое управление | Space |
| Один палец (тач) | Вращение |
| Два пальца (тач) | Зум / панорама |

## 🛠️ Технологии

- **[Three.js](https://threejs.org/)** (r128) — рендеринг сцены, кастомные GLSL-шейдеры
- Чистый JavaScript, без фреймворков и сборщиков
- Web Speech API — распознавание голосовых команд
- Gemini API (опционально, ключ вводится пользователем в браузере) — для сложных голосовых запросов
- `localStorage` — сохранение настроек между сессиями

### Технические заметки
Все «горячие» циклы анимации (нити аккреционного облака, следы орбит гало) написаны с нулевыми аллокациями в кадре — критично для стабильного FPS при сотнях одновременных частиц и линий.
